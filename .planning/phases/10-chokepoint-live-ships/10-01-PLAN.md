    **ChokepointVessel interface:**
    ```typescript
    export interface ChokepointVessel {
      mmsi: string;
      imo: string | null;
      name: string | null;
      flag: string | null;
      shipType: number | null;
      latitude: number;
      longitude: number;
      hasActiveAnomaly: boolean;
      anomalyType: string | null;
    }
    ```

    **getVesselsInChokepoint function:**
    - Accept `chokepointId: string` parameter
    - Look up bounds from `CHOKEPOINTS[chokepointId]` — if not found, return `null` (caller treats as 404)
    - Query using same DISTINCT ON (mmsi) + 1-hour freshness pattern as `countVesselsInChokepoint`
    - LEFT JOIN vessels v ON v.mmsi = vp.mmsi
    - LEFT JOIN vessel_anomalies a ON v.imo = a.imo AND a.resolved_at IS NULL
    - SELECT: vp.mmsi, v.imo, v.name, v.flag, v.ship_type AS "shipType", vp.latitude, vp.longitude, CASE WHEN a.imo IS NOT NULL THEN true ELSE false END AS "hasActiveAnomaly", a.anomaly_type AS "anomalyType"
    - Filter: WHERE vp.latitude BETWEEN bounds.minLat AND bounds.maxLat AND vp.longitude BETWEEN bounds.minLon AND bounds.maxLon
    - ORDER BY vp.time DESC (most recently updated first)
    - Return `ChokepointVessel[]`

    Do NOT modify the existing `countVesselsInChokepoint` or `getChokepointStats` functions — append only.
