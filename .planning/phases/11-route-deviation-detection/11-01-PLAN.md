Implement `geocodeDestination(destination: string): Promise<{ lat: number; lon: number } | null>`:
- Normalize input: `destination.toUpperCase().trim()`
- Return null if empty string after normalization
- Check `geocodeCache` first -- return cached result (including cached nulls) if present
- Fetch `https://nominatim.openstreetmap.org/search?q=${encodeURIComponent(normalized)}&format=json&limit=1`

- On success: parse JSON, if array has results take first result `{ lat: parseFloat(result[0].lat), lon: parseFloat(result[0].lon) }`, cache and return
- On empty results or fetch error: cache null for that destination, return null
- Export the function for testability

Implement `isDeviating(actualHeading: number, expectedHeading: number): boolean`:
- Calculate angular difference: `Math.abs(actualHeading - expectedHeading)`
- If difference > 180, use `360 - difference` (shortest arc)
- Return true if shortest arc > `DEVIATION_THRESHOLD_DEGREES` (45)
- Export the function for testability

Replace the `detectDeviation()` stub with full implementation:
- Query vessels with recent positions in the last 2 hours that have a non-null destination field:
  ```sql
  SELECT v.imo, v.destination,
         array_agg(json_build_object(
           'heading', p.heading,
           'latitude', p.latitude,
           'longitude', p.longitude,
           'time', p.time
         ) ORDER BY p.time) as positions
  FROM vessels v
  JOIN vessel_positions p ON p.mmsi = v.mmsi
  WHERE p.time > NOW() - INTERVAL '2 hours'
    AND v.destination IS NOT NULL
    AND v.destination != ''
    AND p.heading IS NOT NULL
  GROUP BY v.imo, v.destination
  HAVING COUNT(*) >= 2
  ```
- For each vessel:
  1. Call `geocodeDestination(vessel.destination)` -- skip vessel if null
  2. Get the most recent position from the positions array (last element)
  3. Compute expected bearing: `calculateBearing(latestPos.latitude, latestPos.longitude, destCoords.lat, destCoords.lon)`
  4. Check if ALL positions in the array have headings deviating from expected bearing (sustained deviation) using `isDeviating(pos.heading, expectedBearing)` -- note: recompute expectedBearing from each position for accuracy
  5. If all positions deviate (sustained 2+ hours): call `upsertAnomaly` with anomalyType `'deviation'`, confidence `'suspected'`, details matching `DeviationDetails`: `{ expectedHeading: Math.round(expectedBearing), actualHeading: Math.round(latestPos.heading), deviationDegrees: Math.round(angularDiff), destination: vessel.destination }`
  6. If NOT all deviating (heading corrected): call `resolveAnomaly(vessel.imo, 'deviation')` to auto-resolve
  7. Increment count for each new upsert
- Return count

Keep the function signature: `export async function detectDeviation(): Promise<number>`
